# React Native Expo Conversion Prompt

I already have a complete mobile app UI/UX design created in Google Stitch AI and exported as HTML/CSS.

Your task is to convert the existing HTML design into a production-ready React Native Expo application WITHOUT changing the design.

## Critical Requirements

### Design Preservation (Most Important)

* Do NOT redesign anything.
* Do NOT change colors.
* Do NOT change spacing.
* Do NOT change typography.
* Do NOT change animations.
* Do NOT change gradients.
* Do NOT change shadows.
* Do NOT change border radius values.
* Do NOT change screen layouts.
* Pixel-perfect conversion from HTML to React Native.

### Technology Stack

* React Native Expo (latest stable version)
* TypeScript
* Expo Router
* React Native Reanimated v3
* React Native Gesture Handler
* React Native SVG
* Expo Blur
* Expo Linear Gradient
* React Native Safe Area Context
* Zustand for state management
* Axios for API calls

### Conversion Rules

Convert:

HTML div → View

HTML span/p → Text

HTML img → Image

CSS Flexbox → React Native Flexbox

CSS animations → React Native Reanimated

CSS gradients → Expo LinearGradient

CSS shadows → React Native shadow styles

SVG icons → React Native SVG

### Animation Requirements

Replicate ALL animations from the HTML version exactly.

Use:

* Reanimated shared values
* Spring animations
* Smooth transitions
* Gesture interactions
* Loading animations
* Micro interactions
* Page transitions
* Button press effects

Maintain:

* Same duration
* Same easing
* Same motion behavior
