# App Name Research

**Date:** 2026-06-24
**Status:** Resolved — keeping **Aura**
**Reason:** All researched names had domain, trademark, or marketplace conflicts worse than staying with Aura.

## Candidates

### Nosta (from "nostalgia" — warmth of memory)
- **USPTO Class 9/42:** Clear — no live marks
- **App Store:** "Nosta Sauna Lounge" (sauna booking, Health & Fitness)
- **Domains:** `nosta.com` taken (German engineering B2B), `nosta.ai` $29,900 aftermarket, `nosta.app` taken
- **Verdict:** Strongest USPTO position, but domain situation poor

### Lumine (from "lumen" — warm light)
- **USPTO Class 9/42:** "LUMIERE" pending (AI SaaS — phonetic proximity risk), "LUMIN" registered (Magic Leap)
- **App Store:** One web app (creator tools)
- **Domains:** `lumine.app` parked, `lumine.org` taken (1998)
- **Verdict:** Moderate risk from phonetic proximity to LUMIERE

### Velara (invented — warm/velvet + sanctuary)
- **USPTO Class 9/42:** **BLOCKED** — LIVE mark by Velara Inc. (health analytics SaaS, Class 42 + 44)
- **Verdict:** Not available

### Serein (fine rain at sunset — poetic)
- **USPTO:** Has existing mark; is a French word ("serene")
- **App Store:** 3 apps (habit tracker, breathe & reflect, to-do list)
- **Verdict:** Too crowded; dictionary word

### Mellora (from Latin "melior" — better)
- **USPTO:** "Mellora" is an active Norwegian SaaS company (HSEQ+ app)
- **App Store:** Existing HSEQ+ app
- **Verdict:** Existing software company blocks use

## Decision
Staying with **Aura**. A formal attorney clearance is still recommended before US launch.
